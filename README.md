# Simple Receipt Generator

A beginner Python console project that lets the user enter multiple items, prices, and quantities, then prints a simple receipt with subtotals and a grand total.

## Features

- Add multiple items
- Validate quantity input as a whole number
- Validate price input with up to 2 decimal places
- Calculate item subtotals
- Calculate a grand total
- Format money values to 2 decimal places

## Concepts Practiced

- Functions
- While loops
- Input validation
- Lists
- Dictionaries
- For loops
- F-strings
